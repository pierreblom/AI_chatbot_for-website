1. Start System:    python start_simple_system.py
2. Admin Dashboard: http://localhost:5001 (admin/admin123)
3. Add Client:      Dashboard → Clients → Add New
4. Train Bot:       Dashboard → Training → Add Knowledge
5. Generate Code:   Dashboard → Code Generator → Copy HTML
6. Client Integrates: Paste HTML → Chatbot Live!

🎯 Your system perfectly matches the 4-step process: Company → Vector CSV → Training → Integration Code!
