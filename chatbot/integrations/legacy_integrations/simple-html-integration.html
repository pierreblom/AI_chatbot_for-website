This page demonstrates the simplest way to add a chatbot to your website.

Method 1: Auto-Initialize (Easiest)

Just add the script tag with data attributes to your HTML:

<script src="chatbot-widget.js" data-chatbot-api-url="http://localhost:5002" data-chatbot-company-id="your_company_id" data-chatbot-title="Chat with Support" data-chatbot-color="#007bff" data-chatbot-position="bottom-right" data-chatbot-welcome="Hello! How can I help you today?"> </script>

That's it! The chatbot will automatically appear on your page.

Method 2: Manual Initialize (More Control)

Include the script and initialize manually in your code:

<script src="chatbot-widget.js"></script> <script> // Initialize the chatbot const chatbot = new ChatbotWidget('http://localhost:5002', 'your_company_id', { position: 'bottom-right', primaryColor: '#28a745', title: 'Customer Support', welcomeMessage: 'Hi! I\'m here to help you.', placeholder: 'Ask me anything...', showTimestamp: true, enableSoundNotification: false }); </script>

Configuration Options

  • position: 'bottom-right', 'bottom-left', 'top-right', 'top-left'
  • primaryColor: Any valid CSS color (hex, rgb, etc.)
  • title: Title shown in the chat header
  • placeholder: Input field placeholder text
  • welcomeMessage: First message shown to users
  • showTimestamp: Show/hide message timestamps
  • enableSoundNotification: Play sound for new messages
  • maxMessages: Maximum messages to keep in memory

Advanced Control

You can control the chatbot programmatically:

// Clear all messages chatbot.clearMessages(); // Change company (useful for multi-tenant sites) chatbot.setCompanyId('new_company_id'); // Remove the chatbot completely chatbot.destroy();
